# ILibrary#
 
Markup:  *This is a site which eases the process of issuing books.
         *Users can make use of this site to get information about the books present, the time after which it should be submitted.
         *An email would be sent to their gmail accounts to remind them if return date is nearing or has arrived.
         *By going to one's profile, he/she can view the list of all the books issued till date and other information about it.
         *The user's profile would also include all of the books issued till date and related information.
